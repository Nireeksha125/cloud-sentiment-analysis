# Cloud Sentiment Analysis System

This project is a **Cloud-Based Sentiment Analysis System** built using **Natural Language Processing (NLP)** and **Machine Learning (ML)**.

## Features
- Sentiment classification (Positive / Negative)
- Machine Learning model using Logistic Regression
- REST API using Flask
- Interactive Web Interface using Streamlit
- Dataset from Kaggle Sentiment140

## Technologies Used
- Python
- Scikit-learn
- Flask
- Streamlit
- NLP (TF-IDF)

## Project Architecture

User → Streamlit Web App → Flask API → ML Model → Sentiment Prediction

## Dataset
Sentiment140 Dataset from Kaggle
